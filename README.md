## 💧 물막이판 관제 시스템

### 📘 구현 페이지 구조

• 네비게이션 바

📄 home 페이지 ('/home')  
➔ 로그인 기능  
➔ 물막이판 관제 센터 패널  
➔ 각 기능 설명 패널

📄 운영 현황 페이지 ('/dashboard')  
➔ 현 수위 차트 패널 (수위계 차트 시각화)  
➔ 주간 동작 현황 패널 (주간 3시간 단위로 물막이판 상태 표시)  
\* 시간 기준으로 지났을 때 현황 카드 추가

📄 제어 페이지 ('/control')  
➔ 물막이판 제어, 수위계 제어 토글 버튼  
➔ 각 버튼 마다 제어 패널 (3D, 버튼, 슬라이더)  
➔ 현재 상태 요약 패널
